# jsf-microfrontend


## remote React app

cd remote-nodejs-app

npm install

npm start

http://localhost:4000/


## host jsf app

cd host-jsf-app

mvn package

(manually deploy to jboss eap 7.4)

http://localhost:8080/jsf-app-1.0-SNAPSHOT/
